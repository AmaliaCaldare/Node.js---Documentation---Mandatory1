Packages and modules

When developing with Node.js, there are two terms that are frequently used and it is improtant to grasp their meaning from the beginning.

Modules represnt functionalities/ libraries organized in JavaScript files which can be reused throughout a Node.js application.

Packages are used to group together files which offer relevant tools. A package contains modules.

Once you install a package, it will be installed as a dependency together with its modules.

NPM

NPM or Node Package Manager is the default package manager for Node.js.

Node.js instalation comes with the npm command, which allows developers to download and install packages.

When a package is installed in a project, it is considered as being a dependency.

In order to keep track of the dependencies of the project, they have to be listed in a package.json file.

Package.json contains informations about the project and its dependencies and it is essential for both NPM and other end users of the application. It uses the "key": "value" pair format, the key representing a property with its defined value.

Common properties of a package.json file:

  • "name": the name of the package.
  • "version":
  • "main":
  • "repository":
  • "scripts": allows to specify commands that are run at various times in the package

    The "scripts" property is commonly used to define differences between the production and the development side. To run a script type the following command:

                        
                        npm run [script-name]
  • "dependencies": defines the other modules used by the package, by taking their name and the version of each dependency

Creating a new package.json file

Type the following command:

            
                npm init

Answer the questionnaire provived in the command line.

Installing a package

If your project already contains the package.json file, by typing:

        
            npm install

it will store all packages, as dependencies and along with their modules, in the node_modules folder.

If you want to install a specific package, type:

        
            npm install [package-name]

To list the installed packages, you can type:

        
            npm ls

If you want to run a package without installing it, type:

        
            npm install npx
            npx [command]
