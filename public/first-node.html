Steps to create your first Node.js app with Express

  • Plan your Node.js app design and its functionalities

    TIP: Use jspaint.app to prototype.

  • Create a directory to hold your application

    TIP: Think of storing the source code in a git repository.

  • Create the package.json file to describe your project and its properties and install Express

    TIP: Make use of the commands provided by npm.

  • Import the Express module, add the port and start the server.

    TIP: Use the scripts property to specify commands for starting and running the application. Don't forget about nodemon!
