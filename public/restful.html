REST API

Node.js is commonly used for bulding REST based APIs.

What is a REST API?

API stands for application programming interface and represents a set of rules that allow programs to talk with each other over a protocol.

REST determines the architecture of the API, rules followed by developers when creating an API.

It allows two computer systems to communicate over the HTTP protocol, following the client-server principle.

The client makes an HTTP request for a specific URL, the request is routed to a server that will return a response.

URL

The URL is the endpoint or route of a request and it usually contains a domain, port and path.

CRUD and HTTP methods

REST follows four simple HTTP operations which are to CREATE, READ, UPDATE and DELETE data, commonly named CRUD operations.

All the HTTP requests made in a RESTful API map to application CRUD operations and they are handled by the so-called HTTP methods.

HTTP METHOD CRUD Action
GET READ returns requested data
POST CREATE creates new data
PUT UPDATE updates requested data
DELETE DELETE deletes requested data
