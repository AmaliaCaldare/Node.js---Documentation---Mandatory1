Client-side development

Client-side development takes care of the collaboration between the website and the user via the browser.

The goal of the client-side programming is to create an user-friendly and interactive web application, following its specific requirements.

jQuery

jQuery is a JavaScript library used to simplfy the use of JavaScript on a website.

jQuery contains common tasks written in JavaScript, but wrapped in methods that any developer can call.

jQuery usabilities:

  • DOM/HTML manipulation
  • CSS manipulation
  • Effects and animations
  • Events

Adding jQuery to your Web Page

You can download the jQuery library from the official website or include it from a network that hosts it.

In both ways, the source should be included inside the "script" HTML tag.

jQuery Syntax

Select the HTML element and perform an action

             
                $("selector").action();

budo

budo is a browserify server that will live reload/refresh the browser when a HTML, CSS or JavaScript file changes and it provides features for rapid prototyping.

The files will be served on port 9966 and the server will stat with a default index.html file.

To enable live reload on a 'public' directory, type:

        
            budo --dir public/ --live
