JavaScript

Before diving into the Node.js journey, it is important to have a good understanding of the basic JavaScript concepts.

JavaScript is a programming language which uses programs called scripts.

Variables

JavaScript variables are containers used to store values. The variable are declared by using the var or let keywords. Const is also used for variables with constant value.

 
                
                     var firstVariable;
                     let secondVariable;
                     const constantVariable;

Once you declare a variable you can give it a value

                 
                    firstVariable = 'Jon';

Data types

Variables hold values of different data types. JavaScript data types:

  • String: a sequence of text enclosed in quote marks 
                          
                         let name= 'Jon';
  • Number: a number 
                          
                           let age= 20;
  • Boolean: a true/false value, with true or false as keywords 
                         
                        let variable = true;
  • Array: a structure that makes possible to store multiple values in a single variable 
                         
                        let array= ['Jon', 20, 'Rasmus', 30];
                        array[2];  // refer to a specific element of the array
  • Object: collections of related data in the form of key:value pairs 
                         
                        let person = {
                            "name": 'Jon'
                            "age": '20'
                        }

Functions

Functions are used to package a functionality that you want to reuse. When performing a specific task you can call a function, instead of rewriting the code every time.

You can define your own functions or use the ones already built into the browser.

             
                alert("Hello!"); // built-in function

                function addition(x, y) {
                    return x + y;
                }
                addition(2, 6); // defined function
