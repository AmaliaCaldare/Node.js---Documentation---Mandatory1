Introduction to Node.js

What is Node.js?

Node.js is an open-source platform and command-line tool that allows us to execute JavaScript code outside of a browser.

Node.js gives us the possibility to create a web application in a single language, approaching both the server and the client-side development.

With Node.js we can create, read and update files and communicate with databases.

Installing Node.js

A very convenient way to install Node.js is to download the official installer suitable for your operating system, launch it and follow the steps.

When you install Node.js, the node binaries will be added to your path, allowing you to run code.

Running a Node.js script from the command line

To run a Node.js script from the command line you need to use the availabe node command and add the name of the file to be executed.

Make sure to be in the same directory as the node.js file.

               
                   node app.js

TIP: Press CTRL C to stop the programme.
